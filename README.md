# NX-Writer
a way to easily burn bootable drives

### how does it works?
the NX will use a simple CLI (Command-Line Interface) that im working to make it as simple as possible<br>  

to start NX, type
```
NX
```
and it will show a pop-up asking you about which mode you want to use (Simple or Advanced)<br>
you can also type
```
NX --advanced
```
or
```
NX --simple
```
to open it in a specific mode  
plus, you can type
``` 
NX --help
```
to show every command avaiable.
